Решение

Упрощаем Ex​

Сначала разберемся, что из себя представляет запись:

0<∣x∣≤1

Ее можно записать в виде двух одновременно выполняющихся условий:

{0<∣x∣∣x∣≤1​

Будем пользоваться способами упрощения неравенств с модулями (см. прото-задачу П.5).

Для первого неравенства имеем:

0<∣x∣⇔[x>0x<0​

Другими словами, x может быть любым кроме 0, то есть x=0.

Для второго неравенства имеем:

∣x∣≤1⇔−1≤x≤1

Объединяя полученные результаты, x должен удовлетворять следующему условию:

{x=0−1≤x≤1​

Находим Ey​

Имеем функцию котангенса, причем любой x умножается на 4π​, поэтому нам просто нужно найти, какие значения принимает функция ctg при следующих значениях аргумента:

{x=0−4π​≤x≤4π​​

Мы начинаем наш путь от точки x=−4π​:

ctg−4π​=−1

Далее x "движется" в сторону 0. Функция котангенса на этом полуинтервале [−4π​,0) стремится к −∞.

Затем мы "перескакиваем" 0 и движемся к x=4π​. Функция котангенса на этом полуинтервале (0,4π​] принимает значения от +∞ до 1.

Объединяя вместе оба рассуждения выше получаем, что при рассматриваемых x наша функция y принимает все значения, кроме интервала (−1,1).

Ey​=(−∞,−1]∪[1,+∞)

Можно провести действия, обратные действиям из раздела "Упрощаем Ey​". Тогда Ey​ можно записать другим образом:

Ey​=1≤∣y∣<+∞
