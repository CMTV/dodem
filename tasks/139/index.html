Задача 139
Нормальная

Доказать, что если

n→∞lim​xn​=+∞,

то

n→∞lim​nx1​+x2​+…+xn​​=+∞

1
Петр Радько
Зависимость
Решение

По аналогии с предыдущей задачей 138 введем обозначение для последовательности средних геометрических:

ξn​=nx1​+x2​+…+xn​​

Итак, нам известно, что

n→∞lim​xn​=+∞

Распишем это по определению:

∀M>0 ∃N=N(M) ∀n>N : xn​>M

Будем рассматривать какое-то M′>0. Тогда 3M′ тоже положительное и для него существует такое N′, что для любого n>N′:

xn​>3M′

Рассмотрим теперь ξn​ для тех же n:

ξn​=nx1​+x2​+…+xN′​+xN′+1​+…+xn​​

Разделим эту дробь на две:

ξn​=nx1​+…+xN′​​+nxN′+1​+…+xn​​

Рассмотрим дробь справа:

nxN′+1​+…+xn​​

Для каждого слагаемого в числителе выполняется неравенство xi​>3M′, поэтому можно заменить все эти слагаемые на 3M′:

nxN′+1​+…+xn​​>n3M′+3M′+…​=3M′nn−N​

nxN′+1​+…+xn​​>3M′(1−nN​)

Итак:

ξn​=nx1​+…+xN′​​+>3M′(1−nN​)nxN′+1​+…+xn​​​​

ξn​>nx1​+…+xN′​​+3M′(1−nN​)

Найдем предел последовательности в скобках справа (принимая во внимание, что N′ — константа):

n→∞lim​(1−nN′​)=1−N′n→∞lim​n1​=1−N′⋅0=1

Мы воспользовались тем, что n1​→0 (см. прото-задачу П-ссылка).

Мы показали, что

n→∞lim​1−nN′​=1

По определению это значает, что

∀ε>0 ∃N=N(ε) ∀n>N : ∣∣​1−nN​−1∣∣​<ε

Раз выполняется для любого положительного ε, то выполняется и для 21​. Итак, для 21​ существует такое N′′, что для любого n>N′′ выполняется неравенство:

∣∣​1−nN′​−1∣∣​<21​

Упростим это неравенство с модулем по пункту 1 прото-задачи П-ссылка:

∣∣​1−nN′​−1∣∣​<21​⇔{1−nN′​−1<21​1−nN′​−1>−21​​

Рассмотрим нижнее неравенство справа:

1−nN′​−1>−21​

Прибавим к обеим частям 1:

1−nN′​>21​

Возвращаемся к нашему неравенству с ξn​:

ξn​>nx1​+…+xN′​​+3M′>21​(1−nN′​)​​

ξn​>nx1​+…+xN′​​+23M′​

Стоит заметить, что здесь речь уже идет о n>max(N′,N′′), чтобы выполнялись сразу оба неравенства из определений для xn​→+∞ и (1−nN′​)→1.

Рассмотрим теперь дробь

nx1​+…+xN′​​

Найдем предел этой последовательности (принимая во внимание, что x1​+…+xN′​ — константа):

n→∞lim​nx1​+…+xN′​​=(x1​+…+xN′​)n→∞lim​n1​=0

По определению это означает, что

∀ε>0 ∃N=N(ε) ∀n>N : ∣∣​nx1​+…+xN′​​∣∣​<ε

Раз выполняется для любого положительного ε, то и для числа 2M′​ найдется такое N′′′, что для любого n>N′′′ будет выполняться неравенство

∣∣​nx1​+…+xN′​​∣∣​<2M′​

Упростим это неравенство с модулем по пункту 1 прото-задачи П-ссылка:

∣∣​nx1​+…+xN′​​∣∣​<2M′​⇔{nx1​+…+xN′​​<2M′​nx1​+…+xN′​​>−2M′​​

Рассмотрим нижнее неравенство справа:

nx1​+…+xN′​​>−2M′​

Прибавим к обеим частям 23M′​:

nx1​+…+xN′​​+23M′​>−2M′​+23M′​

nx1​+…+xN′​​+23M′​>M′

Итак,

ξn​>nx1​+…+xN′​​+23M′​>M′

ξn​>M′

Стоит заметить, что здесь речь уже идет о n>max(N′,N′′,N′′′), чтобы выполнялись все три неравенства из определений для xn​→+∞, (1−nN′​)→1 и nx1​+…+xN′​​→0.

Мы показали, что какое бы положительное число M мы не взяли, всегда найдутся три таких N′,N′′,N′′′, что для любого n>N=max(N′,N′′,N′′′) будет выполняться неравенство:

ξn​=nx1​+…+xn​​>M

nx1​+x2​+…+xn​​>M

Это по определению означает, что

n→∞lim​nx1​+x2​+…+xn​​=+∞
