§1Вещественные числаВерхние и нижние грани

Пусть {xy} есть множество всех произведений xy, где x∈{x} и y∈{y}, причем x≥0 и y≥0. Доказать равенства:

а) inf{xy}=inf{x}inf{y};б) sup{xy}=sup{x}sup{y}

Решение

Докажем сначала важное свойство. Если числовое множество {x} состоит из неотрицательных чисел, то inf{x}≥0 и sup{x}≥0.

Запишем, что означает, что {x} состоит из неотрицательных чисел:

∀x∈{x} : x≥0

Из этой записи по определению следует, что 0 — нижняя граница {x}. Если 0 и является точной нижней границей, то все нормально. Если нет, то точная нижняя граница будет, по определению, больше 0.

Итак, мы доказали, что если {x} состоит из неотрицательных чисел, то inf{x}≥0.

Точная верхняя грань не меньше любого элемента из {x}, а потому она не меньше, чем inf{x}:

sup{x}≥∀x∈{x}≥inf{x}≥0

Доказательство а)

inf{xy}=inf{x}inf{y}

Нужно доказать два пункта:

1) ∀(xy)∈{xy} : (xy)≥inf{x}inf{y}

2) ∀ε>0 ∃(x′y′)∈{xy} : (x′y′)<inf{x}inf{y}+ε

Доказательство п.1)

Докажем от противного:

∃(xy)∈{xy} : (xy)<inf{x}inf{y}

То есть, существует такое произведение (xy), которая меньше произведения инфинумов множеств {x} и {y}. Раз существует произведение, то существуют и составляющие его множители: x и y.

Из определения inf{x}:

x≥inf{x}

Из определения inf{y}:

y≥inf{y}

Уножим друг на друга оба неравенства (мы можем их умножить, так как inf{x}≥0 и inf{y}≥0, как мы показали в начале решения):

xy≥inf{x}inf{y}

Получаем два противоречащих друг другу неравенства:

\begin{gather*} xy \geq \inf\set{x}\inf\set{y} \[5px] xy < \inf\set{x}\inf\set{y} \end{gather*}

То есть, xy одновременно меньше и не меньше произведения инфинумов. Противоречие.

Значит, выполняется обратное, то есть

∀(xy)∈{xy} : (xy)≥inf{x}inf{y}

■

Доказательство п.2)

Докажем от противного:

∃ε′>0 ∀(x′y′)∈{xy} : (x′y′)≥inf{x}inf{y}+ε′

То есть, существует такое ε′>0 что произвольное произведение x′y′ не будет меньше произведения инфинумов, плюс ε′. Раз мы рассматриваем произвольное произведение (x′y′), то есть какие-то множители x′∈{x} и y′∈{y}, из которых оно состоит.

Из определения inf{x}:

∀ε>0 ∃x′<inf{x}+ε

Так как это выражение работает для любого ε, то оно работает и для числа inf{x}+inf{y}inf{x}ε′​

То есть, для положительного числа выше

∃x′′<inf{x}+inf{x}+inf{y}inf{x}ε′​

Из определения inf{y}:

∀ε>0 ∃y′<inf{y}+ε

Так как это выражение работает для любого ε, то оно работает и для числа inf{x}+inf{y}inf{y}ε′​

То есть, для положительного числа выше

∃y′′<inf{y}+inf{x}+inf{y}inf{y}ε′​

Итак, имеем два неравенства:

x′′<inf{x}+inf{x}+inf{y}inf{x}ε′​

y′′<inf{y}+inf{x}+inf{y}inf{y}ε′​

Умножим эти два неравенства друг на друга:

x′′y′′<inf{x}inf{y}+inf{x}+inf{y}inf{x}ε′+inf{y}ε′​+(inf{x}+inf{y})2inf{x}inf{y}ε′2​

x′′y′′<inf{x}inf{y}+ε′+(inf{x}+inf{y})2inf{x}inf{y}ε′2​

Неравенство можно усилить, отбросив последнее положительное слагаемое:

x′′y′′<inf{x}inf{y}+ε′

По предположению от противного,

∀(x′y′)∈{xy} : (x′y′)≥inf{x}inf{y}+ε′

Раз выполняется для всех сумм, то выполняется и для произведения x′′y′′, поэтому получаем два противоречащих друг другу неравенства:

\begin{gather*} x''y'' \geq \inf\set{x}\inf\set{y} + \varepsilon' \[5px] x''y'' < \inf\set{x}\inf\set{y} + \varepsilon' \end{gather*}

Произведение x′′y′′ получается одновременно меньше и не меньше произведения инфинумов, плюс ε′. Противоречение.

Значит, выполняется обратное, то есть

∀ε>0 ∃(x′y′)∈{xy} : (x′y′)<inf{x}inf{y}+ε

■

Итак, по пункту 1) произведение inf{x}inf{y} является нижней гранью множества {xy}, а по пункту 2) оно является точной нижней гранью:

inf{xy}=inf{x}inf{y}

■

Доказательство б)

sup{xy}=sup{x}sup{y}

Доказательство точно такое же, как и доказательство а), приведенное выше, только доказать надо следующие два пункта:

1) ∀(xy)∈{xy} : (xy)≤sup{x}sup{y}

2) ∀ε>0 ∃(x′y′)∈{xy} : (x′y′)>sup{x}sup{y}−ε
