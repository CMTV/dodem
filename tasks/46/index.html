Указание

Вынести число 10000 за предел. Оставшуюся часть ограничить сверху последовательностью n1​, которая стремится к 0 (см. прото-задачу П.4).

Решение

Вынесем константу 10000 из предела:

n→∞lim​n2+110000n​=10000⋅n→∞lim​n2+1n​

Докажем, что

n2+1n​<?​n2n​=n1​

Умножаем обе части на n⋅(n2+1):

n2<?​n2+1

Вычитаем из обеих частей n2:

0<1

Итак, мы доказали, что:

n2+1n​<n1​

Значит, нашу последовательность из условия можно "зажать":

0<n2+1n​<n1​

Предел 0 при n→∞ равен 0. Предел n1​ при n→∞ тоже равен 0 (см. прото-задачу П.4).

По теореме о двух милиционерах это означает, что "зажатая" последовательность из условия тоже стремится к 0:

n→∞lim​n2+1n​=0

Значит:

n→∞lim​n2+110000n​=10000⋅n→∞lim​n2+1n​=10000⋅0=0
