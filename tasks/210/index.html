Указание

Вручную найдите, чему равно f(f(x)) и f(f(f(x))). По результатам сразу найдете нужную формулу.

Докажите формулу по методу математической индукции.

Решение

Исследование

f1​(x)=1+x2​x​

f2​(x)=1+f(x)2​f(x)​=1+1+x2x2​​1+x2​x​​=1+x2​1+x2​1+2x2​​x​=1+2x2​x​

Закономерность заметили: при увеличении n увеличивается на единицу коэффициент при x2 в знаменателе.

Доказательство

Докажем, что

fn​(x)=1+nx2​x​

Доказывать будем по методу математической индукции.

База индукции:

Пусть n=1, тогда:

f1​(x)=f(x)=1+x2​x​

Индукционный переход:

Пусть равенство выполняется для какого-то натурального k:

fk​(x)=1+kx2​x​

Рассмотрим, чему равняется fk+1​(x):

fk+1​(x)=f(fk​(x))=1+fk​(x)2​fk​(x)​==1+1+kx2x2​​1+kx2​x​​=1+kx2​1+kx2​1+(k+1)x2​​x​=1+(k+1)x2​x​

Итак:

fk+1​(x)=1+(k+1)x2​x​

Индукционный переход доказан. Это означает, что доказываемая формула работает для любого натурального n.
