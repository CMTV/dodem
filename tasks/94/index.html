Решение

Обозначим последовательность в условии за xn​. Известно, что она сходится к какому-то числу A.

Предположим, что

∀n : xn​=A

То есть, каждый член xn​ равен A. В этом случае, очевидно, что

xn​=supxn​=infxn​=A

Рассмотрим теперь отрицание утверждения выше:

∃n′ : xn′​=A

Обозначим xn′​ за B. Будем считать, что A>B (для B>A доказательство аналогичное).

Распишем по определению, что значит xn​→A:

∀ε>0 ∃N=N(ε) ∀n>N : ∣xn​−A∣<ε

Раз выполняется для любого положительного ε, то и для числа 2A−B​ найдется такое N, что для любого n>N выполняется неравенство:

∣xn​−A∣<2A−B​

Важно отметить, что N≥1. Пусть это не так и N=0. Тогда для абсолютно всех членов последовательности xn​ выполняется неравенство

∣xn​−A∣<2A−B​

В том числе оно выполняется и для xn′​, то есть

∣xn′​−A∣<2A−B​

Но xn′​=B, поэтому

∣xn′​−A∣<2A−B​

Так как A>B, то B−A<0, поэтому

∣xn′​−A∣=∣B−A∣=−(B−A)=A−B

A−B<2A−B​

1<21​

Получили противоречие. Значит N не может быть равно 0, поэтому N≥1. Но N — конечное число и мы имеем N первых элементов последовательности xn​:

x1​,x2​,…,xN​

Среди этого конечного числа членов есть хотя бы один, а в общем случае некоторое конечное множество элементов {xt​}, для которых не выполняется неравенство из определения предела, то есть

∣xt​−A∣≥2A−B​

Распишем это неравенство по пункту 2 прото-задачи П.18:

∣xt​−A∣≥2A−B​⇔[xt​−A≥2A−B​xt​−A≤−2A−B​​

Пусть верно первое неравенства справа (случай, когда верно второе доказывается аналогично):

xt​−A≥2A−B​

xt​≥A+2A−B​

Мы знаем, что для любого n>N выполняется неравенство

∣xn​−A∣<2A−B​

Распишем это неравенство по пункту 1 прото-задачи П.18:

∣xn​−A∣<2A−B​⇔{xn​−A<2A−B​xn​−A>−2A−B​​

Рассмотрим первое неравенство справа:

xn​−A<2A−B​

A+2A−B​>xn​

Теперь воспользуемся неравенством для xt​ выше:

xt​>A+2A−B​>xn​

xt​>xn​

Итак, полученный член xt​ больше всех членов последовательности, начиная с n>N. Если есть и другие элементы из {xt​}, для которых тоже выходит неравенство выше, то берем максимальный из них.

Итак, получили некоторый элемент x, который больше любого xn​ при n>N и одновременно больше любого из конечного числа элементов x1​…xN​. Значит x больше вообще любого члена последовательности xn​. Но так как x является членом последовательности xn​, значит

x=supxn​

То есть последовательность xn​ достигает своей точной верхней грани.
