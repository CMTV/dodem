Задача 225
Задача 225
Нормальная
Определить обратную функцию x=ϕ(y) и ее область существования, если:

y=x2; а) −∞<x⩽0;  б) 0⩽x<+∞

Ответ

Пункт а)

x=ϕ(y)=−y​(y≥0)

Пункт б)

x=ϕ(y)=y​(y≥0)

1
Петр Радько
Указание

Воспользуйтесь свойством модуля из прото-задачи П-ссылка:

f2​=∣f∣

Решение

y=x2

Возьмем квадратный корень от обеих частей равенства:

∣x∣=y​

Взяв квадратный корень из y, мы должны ограничить y только неотрицательными значениями, то есть y≥0.

При значениях x из пункта а) ∣x∣=−x, поэтому:

−x=y​

x=ϕ(y)=−y​

При значениях x из пункта б) ∣x∣=x, поэтому:

x=ϕ(y)=y​
