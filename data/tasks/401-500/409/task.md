Пусть

$$ R(x) = \frac{a_0x^n + a_1x^{n-1} + \ldots + a_n}{b_0x^{m}+b_1x^{m-1}+\ldots+b_m}, $$

где $a_0 \neq 0$ и $b_0 \neq 0$.

Доказать, что

$$ \lim_{x \to \infty} R(x) = \andcases{ \infty, &\text{если } \ n > m; \\[5px] \dfrac{a_0}{b_0}, &\text{если } \ n=m; \\[8px] \phantom{.}0, &\text{если } \ n < m. } $$
