## Пункт а)

$$ \lim_{n\to\infty} = \infty \Leftrightarrow \forall E > 0 \ \exists N = N(E) \ \forall n > N \ : \ |x_n| > E $$

## Пункт б)

$$ \lim_{n\to\infty} = +\infty \Leftrightarrow \forall E > 0 \ \exists N = N(E) \ \forall n > N \ : \ x_n > E $$

## Пункт в)

$$ \lim_{n\to\infty} = -\infty \Leftrightarrow \forall E > 0 \ \exists N = N(E) \ \forall n > N \ : \ x_n < -E $$
