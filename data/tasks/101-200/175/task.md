Функция $y = \sgn x$ определяется следующим образом:

$$
\sgn x =
\begin{dcases}
-1, &\text{если } x < 0;
\\
\phantom{-}0, &\text{если } x = 0;
\\
\phantom{-}1, &\text{если } x > 0.
\end{dcases}
$$

Построить график этой функции. Показать, что

$$ |x| = x \sgn x. $$
