Предел сложной функции

Удобная формула для вычисления пределов с помощью замены переменной.
Предел сложной функции

Если функция f(x) имеет в точке a предел b и не равна b в некоторой проколотой окрестности точки a, а функция g(y) имеет в точке b предел c, то сложная функция g(f(x)) имеет предел в точке a и он равен c.

∃x→alim​f(x)=b∃ U˚(a) : b∈f(U˚(a))∃y→blim​g(y)=c​⎭⎬⎫​ ⇒ ∃x→alim​g(f(x))=c

Замечание: теорема работает для конечных и бесконечных пределов.

Доказательство

Рассмотрим произвольную последовательность {xn​}⊂U˚(a), которая стремится к a. Согласно определению предела функции по Гейне (П-ссылка), соответствующая ей последовательность {f(xn​)}→b, причем по условию ни один ее член {f(xn​)} не равен b.

Обозначим {f(xn​)} за {yn​}. Так как ни один из ее членов не равен b, мы можем еще раз воспользоваться определением предела функции по Гейне из которого следует, что последовательность {g(yn​)}→c.

Резюмируя. Мы взяли произвольную последовательность {xn​}⊂U˚(a), которая стремится к a и показали, что соответствующая ей последовательность значений функции g(f(x)) стремится к c. Это по определению предела функции по Гейне означает, что

∃x→alim​g(f(x))=c

■

Эта теорема позволяет выполнять замену переменных при вычислении пределов сложных функций по формуле:

x→alim​g(f(x))=y→blim​g(y)
