Количество предельных точек

Если A1​,A2​,…Ak​ — предельные точки последовательности xn​ и каждый член xn​ есть хотя бы в одной из подпоследовательностей y1n​,y2n​,…ykn​, таких, что

n→∞lim​y1n​=A1​n→∞lim​y2n​=A2​⋯n→∞lim​ykn​=Ak​

то других предельных точек у последовательности xn​ нет.

Решение

Будем доказывать от противного. Пусть у последовательности xn​ существует еще одна отличная от всех других предельная точка At​. По определению это означает, что существует некоторая подпоследовательность ytn​, такая что

n→∞lim​ytn​=At​

Рассмотрим число E, равное

E=3min∣Ai​−Aj​∣​, (i,j=1,2,…k,t)

В числителе мы находим такие две предельные точки, расстояние между которыми минимально (включая в рассмотрение "якобы" предельную точку At​). Затем найденное минимальное расстояние делим на 3, чтобы исключить пересечение E-окрестностей для разных предельных точек.

Распишем по определению, что означает предел подпоследовательностей y1n​,y2n​,…ykn​,ytn​:

n→∞lim​y1n​=A1​⇔∀ε>0 ∃N1​ ∀n>N1​ : ∣y1n​−A1​∣<εn→∞lim​y2n​=A2​⇔∀ε>0 ∃N2​ ∀n>N2​ : ∣y2n​−A2​∣<ε⋯n→∞lim​ykn​=Ak​⇔∀ε>0 ∃Nk​ ∀n>Nk​ : ∣ykn​−Ak​∣<εn→∞lim​ytn​=At​⇔∀ε>0 ∃Nt​ ∀n>Nt​ : ∣ytn​−At​∣<ε

Раз все определения выполняются для любого положительного ε, то они выполняются и для ранее полученного числа E.

Пусть

N=max(N1​,N2​,…,Nk​,Nt​)

Итак, для E существует такое N, что для любого n>N одновременно выполняются неравенства:

∣y1n​−A1​∣<E∣y2n​−A2​∣<E⋯∣ykn​−Ak​∣<E∣ytn​−At​∣<E

Каждое из этих неравенств означает, что член подпоследовательности yin​ лежит в E-окрестности точки Ai​:

Рассмотрим последнее неравенство. Для E существует такое N, что для любого n>N выполняется

∣ytn​−At​∣<E

Эта запись означает, что член подпоследовательности ytn​ лежит в E-окрестности точки At​. Этот член подпоследовательности является членом исходной последовательности с каким-то номером m:

ytn​=xm​

В условии сказано, что каждый член исходной последовательности xn​ принадлежит одной из подпоследовательностей y1n​,y2n​,…,ykn​. Значит и ytn​=xm​ принадлежит одной из этих подпоследовательностей. Предположим, он принадлежит подпоследовательности y4n​. Это означает, что ytn​ лежит в E-окрестности точки A4​, а не точки At​.

Подобные рассуждения можно провести для любого члена ytn​ при n>N. Это означает, что после N никакой элемент ytn​ не находится в E-окрестности точки At​. Получили противоречие. Значит At​ — не может быть предельной точкой последовательности xn​.
