Решение

Доказывать будем от противного. Пусть

A>B

По условию известно, что

n→∞lim​xn​=A

По определению предела это означает, что

∀ε>0 ∃N=N(ε) ∀n>N : ∣xn​−A∣<ε

Это означает, что для любого положительного ε найдется такое N, что для любого следующего за N номера выполняется неравенство ∣xn​−A∣<ε.

Значит и для положительного числа 2A−B​ найдется такое Nx​, что для всех членов xn​ за ним будет выполняться неравенство:

∣xn​−A∣<2A−B​

Проводя аналогичные рассуждения для предела yn​, найдется Ny​ такое, что для всех членов yn​ за ним будет выполняться неравенство

∣yn​−B∣<2A−B​

Выбираем максимальное из Nx​ и Ny​. Для такого максимального Nmax​ будут одновременно выполняться два неравенства:

∣xn​−A∣<2A−B​

∣yn​−B∣<2A−B​

Сложим оба неравенства друг с другом:

∣xn​−A∣+∣yn​−B∣<A−B

Воспользовавшись свойствами модулей, усилим неравенство:

∣yn​−xn​+(A−B)∣≤∣A−xn​∣+∣yn​−B∣<A−B

∣yn​−xn​+(A−B)∣<A−B

Пояснение усиления

Итак, имеем ∣xn​−A∣+∣yn​−B∣<A−B

Есть свойство модулей:

∣f−g∣=∣g−f∣

Используем это свойство для первого слагаемого в неравенстве выше:

∣A−xn​∣+∣yn​−B∣<A−B

Есть еще одно свойство модулей:

∣f+g∣≤∣f∣+∣g∣

Воспользуемся этим свойством для усиления неравенства:

∣yn​−xn​+(A−B)∣≤∣A−xn​∣+∣yn​−B∣<A−B

∣yn​−xn​+(A−B)∣<A−B

Разобьем это неравенство на два по прото-задаче П.18:

Рассмотрим первое неравенство:

yn​−xn​+(A−B)<A−B

Вычтем из обеих частей A−B:

yn​−xn​<0

yn​<xn​

Но по условию xn​≤yn​. Итак, получаем два противоречащих друг другу неравенства:

xn​≤yn​xn​>yn​
