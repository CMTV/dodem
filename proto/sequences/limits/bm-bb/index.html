Решение

Докажем в одну сторону: если αn​→0, то αn​1​→∞. Запишем развернуто:

n→∞lim​αn​=0⇒n→∞lim​αn​1​=∞

Сначала выразим словами, как это будет доказываться. Мы из известного определения левого предела строим определение для правого.

Распишем по определению левый предел:

n→∞lim​αn​=0⇔∀ε>0 ∃N=N(ε) ∀n>N : ∣αn​∣<ε

Раз выражение выше работает для любого ε>0, то оно будет работать и для положительного числа E1​, то есть:

∀E>0 берем ε′=E1​

Теперь, это ε′ мы подставляем в определение выше, так как оно справедливо для всех положительных чисел.

Из него мы и получаем следующее:

∃N=N(ε′) ∀n>N : ∣αn​∣<ε′

Разберемся с последним неравенством:

∣αn​∣<ε′=E1​⇔∣∣∣∣∣​αn​1​∣∣∣∣∣​>E

Объединим все вместе "на словах". Из любого E>0 мы берем E1​>0 и подставляем его в определение выше. Оттуда получаем N, такое, что для любого n>N будет выполнятся неравенство:

∣∣∣∣∣​αn​1​∣∣∣∣∣​>E

Теперь запишем это же формально:

∀E>0 ∃N=N(E1​) ∀n>N : ∣∣∣∣∣​αn​1​∣∣∣∣∣​>E

А это и есть определение бесконечно большой последовательности αn​1​:

n→∞lim​αn​1​=∞
